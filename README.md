# ML Project with Marimo

## Описание
Интерактивный интерфейс для ML модели.

## Запуск
```bash
pip install -r requirements.txt
marimo run hw_simple_new.py
```

## Данные
Файлы данных не включены в репозиторий из-за большого размера.
Для работы модели требуются:
- `flight_data_2024.csv`
- `flight_data_processed.csv`

```bash
# Установка зависимостей
```
